# PoolKimi

A premium 3D interactive pool configurator with real-time pricing based on 2025 market data.

## Features

- **3D Pool Viewer** - Hyper-realistic Three.js pool model that updates in real-time
- **Interactive Configuration** - Choose shapes, finishes, water colors, deck materials, and premium features
- **Real-Time Pricing** - Accurate USD price estimates based on national market data
- **Gallery** - Browse 12 pre-designed pool configurations for inspiration
- **Responsive Design** - Works beautifully on desktop, tablet, and mobile

## Tech Stack

- React 19 + TypeScript + Vite
- Three.js + React Three Fiber (3D rendering)
- Tailwind CSS + shadcn/ui
- GSAP (animations)
- React Router (routing)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
